Frequently Asked Questions

Have a question not answered here? Create a ticket in our Discord Server

When is the client going to release?


The client has already been released and is currently in Open Beta. Anyone can download and use the client, however please be aware that there will be issues and potential crashes.

What operating systems are supported?


We currently support 64-bit Windows, macOS and Linux.


If our download page is missing a Linux format that is compatible with your distribution, please get in touch with us and we will see what we can do.


32-bit operating systems will never be supported as Minecraft itself no longer supports them as of 1.17.

What Minecraft versions does the client support?


We currently support all of the versions beyond 1.18 as well as legacy versions such as 1.8 & 1.12. Newer versions will be supported as they release.


We may drop support for any versions not deemed long-term-supported by mod developers.

Do you support "cracked" or "offline" accounts?


While we do allow offline/cracked login, it can only be accessed if you have already logged in with a premium account. This will never be changed.

Are you able to add your own mods?


Yes, you can simply Drag and Drop the mods in the "Mods" section of the launcher.


Keep in mind that it is your responsibility to ensure the mods are compatible with the version you are trying to run.
